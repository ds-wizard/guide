Getting Started
***************

This section is a practical entry point for people who want to get value from |project_name| quickly. Instead of describing screens one by one, it follows the most common jobs: understanding the core building blocks, preparing a project, creating reusable stewardship content, and running a local instance.

Choose the path that matches what you need right now:

- New user or evaluator: start with :ref:`dsw-core-concepts`.
- Researcher preparing a DMP: continue with :ref:`researcher-workflow`.
- Data steward designing reusable guidance: continue with :ref:`data-steward-workflow`.
- Administrator setting up or validating an instance: continue with :ref:`admin-quickstart` and :ref:`local-deployment-quickstart`.
